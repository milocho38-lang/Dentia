# D000 - Visión del Producto

Nombre provisional: Dentia

Objetivo:
Desarrollar una plataforma integral para odontólogos,
consultorios y clínicas odontológicas.

Usuarios:

- Odontólogo independiente
- Secretaria
- Administrador
- Clínica odontológica

MVP:

- Pacientes
- Agenda
- Historia clínica
- Odontograma
- Presupuestos
- Tratamientos
- Pagos
- Caja

Tecnologías:

Frontend:
- Next.js
- TypeScript
- Tailwind

Backend:
- FastAPI
- SQLAlchemy
- PostgreSQL

Principios:

- Multiempresa
- Multiusuario
- Preparada para SaaS
- Preparada para IA
- Fácil de usar
