# D006 - Identidad Visual

Versión: 1.0

## Nombre Comercial

Dentia

Dentia queda confirmado como nombre oficial y definitivo del producto.

---

## Slogan Corporativo

El slogan corporativo oficial de Dentia es:

```text
Gestión Odontológica Inteligente
```

Este slogan reemplaza cualquier formulación corporativa anterior.

---

## Concepto de Marca

Dentia es una plataforma moderna para la gestión integral de consultorios y clínicas odontológicas.

La marca debe transmitir:

* Confianza
* Salud
* Organización
* Profesionalismo
* Tecnología
* Cercanía

---

## Estilo Visual

Clínica Moderna

El estilo Clínica Moderna queda confirmado como estándar visual oficial de Dentia.

Características:

* Diseño limpio.
* Mucho espacio visual.
* Interfaz clara.
* Navegación sencilla.
* Apariencia profesional.
* Orientada a personal administrativo y clínico.

Evitar:

* Interfaces oscuras.
* Exceso de colores.
* Diseño tipo gamer o tecnológico agresivo.
* Elementos visuales recargados.
